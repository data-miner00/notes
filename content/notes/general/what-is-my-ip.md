The endpoints that can be used to retrieve public IP of a device.

## Endpoints

```
curl icanhazip.com
```

```
curl ifconfig.me/ip
```

```
curl api.ipify.org
```

```
curl ipinfo.io/ip
```

```
curl ipecho.net/plain
```

```
curl whatismyip.akamai.com
```

```
curl wtfismyip.com/text
```

```
nslookup myip.opendns.com resolver1.opendns.com
```

### IP with details

```
curl ipinfo.io
```
